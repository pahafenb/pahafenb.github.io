Questions & Answers

Question#8
How would you create a simple two-row, two column table with a standard border?

To create this type of table, this format/code may be used and text can be inserted into the slots where it says “TextHere”. Adding a standard border comes without adding coding.

<table>
<tr>
<th>TextHere</th>
<th>TextHere</th>
</tr>
<tr>
<td>TextHere</td>
<td>TextHere</td>
</tr>
</table>

Question #9
How would you add 30 pixels of space between the table border and the cells?

In HTML:

<table class=“border”>
<tr>
<th>TextHere</th>
<th>TextHere</th>
</tr>
<tr>
<td>TextHere</td>
<td>TextHere</td>
</tr>
</table>

In CSS:

table.border{
    border-spacing:30px;
}

Question #10

How would you make the top-left cell green, the top-right cell red, the bottom left cell yellow and the bottom right cell blue?

In HTML:

<table>
<tr>
<th class="green">This Text is Green</th>
<th class="red">This Text is Red</th>
</tr>
<tr>
<td class="yellow">This Text Is Yellow</td>
<td class="blue">This text is Blue</td>
</tr>
</table>

In CSS:

th.green {
    background-color: green;
}
th.red {
    background-color:red;
}
td.yellow {
    background-color:yellow;
}
td.blue {
    background-color:blue;
}

Question #11
How would you give a web page a black background and make all text bright green?

You may do this by adding the style background-color and color into the body tag as shown:
<body style="background-color:black;color:green">

Question #12
If you specify background-color #FFFFFF for the body element in your style sheet, then use background-color #FF0000 for the first <div> that you use on your page, will the background of that <div> be red or white? Why?

Red will show up as specific tags unlike a general body tag takes priority. This makes it possible so if you want the whole page to be let us say white but certain sections or tags to be another color/colors.

Question #13
To place two <div> elements next to each other, but with a 30pixel margin between them, what entry or entries can you use in the style sheet?

If both elements are next to each other, a margin property may be added to the tag. This is shown by margin-left, margin-right, margin-top, or margin-bottom. Then the amount of space may be added. So if I want the space between the two to be 50 pixels, I can add in CSS:

div.leftdiv {
    margin-right:25px
}
div.rightdiv {
    margin-left:25px
}

Question #14
Which CSS style property and value is used to ensure that content does not appear to the left of a floating element?

To ensure content does not appear to the left of the floating element, this code may be used in CSS:

name{
    float: left;
}

Question #15
What style sheet entry is used to place text within a <div> to appear 12 pixels from the top of the element?

To do this, you may use this code in the HTML <div> tag:

div {
    padding-top:12px;
}

Question #16
What's the difference between relative positioning and absolute positioning?

Relative positioning positions an element in relation to another. So for example, if you wanted to position a heading 15 pixels above the first paragraph. Absolute positioning measures the exact coordinates of where the element will go regardless if there is already something in that spot.

Question #17
Which CSS style property controls the manner in which elements overlap each other?

The z-index controls which order the elements overlap. The higher the number, the higher the priority which means that it will show up on top of any lower ones.

Question #18
What HTML code could you use to display the words “Where would you like to” starting exactly at the upper-left corner of the browser window and displays the words “Go Today?” in large type exactly 80 pixels down and 20 pixels to the left of the corner?

<p style="position: fixed;left:0px;top:0px">Where would you like to</p>
<h1 style="position: fixed;left:20px;top:80px;">Go Today?</h1>
